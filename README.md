# Revelry 🎉

Modern party games for friends, families, and teams.

## Features

- Browser-based
- Real-time multiplayer
- Mobile friendly
- AI-powered game generation
- No downloads required

## Tech Stack

- Next.js
- Vue.js
- FastAPI
- tornado
- PostgreSQL
- Redis
- WebSockets
